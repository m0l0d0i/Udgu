Работа с массивом строк V[9]

Задание:

Создать программу, формирующую массив строк V[9] и осуществляющую с ним следующие действия:

  1. Вывод массива на экран
  2. Вывод количества элементов массива, в которых встречается сочетание 'c++' (регистр не имеет значения)
  3. Вывод на экран первых четырех символов 6-го элемента массива
  4. Вывод на экран последних трех символов элемента № 5
